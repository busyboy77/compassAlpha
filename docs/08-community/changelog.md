# Changelog

> What changed in CompassAlpha, version by version. Newest first.

This changelog records notable changes to the framework and its documentation portal. It groups changes under **Added**, **Changed**, **Deprecated**, **Removed**, **Fixed**, and **Security** as they apply. Versions follow the framework's own release line; see the [Roadmap](roadmap.md) for what each milestone means.

---

## v0.1 DRAFT — 2026-06

The first published form of CompassAlpha.

### Added

- Initial **source-available** release under the [Business Source License 1.1](https://github.com/busyboy77/compassAlpha/blob/main/LICENSE), published by [gradus](https://gradus.pk).
- **Foundation** section of the portal — framework-not-tool framing, constitution overview, the GitAI category, and the glossary.
- **Axioms** section — the framework's invariant constitution.
- Licensing surface: [LICENSE](https://github.com/busyboy77/compassAlpha/blob/main/LICENSE), [COMMERCIAL.md](https://github.com/busyboy77/compassAlpha/blob/main/COMMERCIAL.md), and NOTICE at the repository root.

### Changed

- The framework was **extracted into a clean, abstract, adopter-independent form** — separated from any specific implementation so it stands on its own as a reusable constitution, set of conventions, and worked examples.
