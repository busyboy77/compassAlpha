## Why this section exists

CompassAlpha is unusual in its category — it's neither a tool you install nor a service you subscribe to. Adopters routinely mistake it for one of those in the first 5 minutes and bounce. This section disambiguates immediately.

If you only read **one page** in this entire portal, read [Framework, not tool](framework-not-tool.md). Everything else follows from getting that framing right.
